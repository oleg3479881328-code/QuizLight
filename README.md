# QuizLight

QuizLight is a lightweight learning platform focused on fast flashcard study, simple quizzes, and lightweight knowledge review.

## Current Stack

- Vite
- React
- TypeScript
- local-first browser persistence planned for `v0`

## Current Status

- first functional MVP implemented
- Russian-English card CRUD with local persistence
- two-sided card flip preview with speech playback
- dark/light theme toggle with system preference detection
- image URL support on cards with display in preview and collection
- Google Image Search buttons (🇷🇺🔍 / 🇬🇧🔍) in the editor
- running on Vite dev server (port 5174)

## Scripts

- `npm run dev` - start the development server
- `npm run build` - create a production build
- `npm run lint` - run ESLint
- `npm run preview` - preview the production build locally
